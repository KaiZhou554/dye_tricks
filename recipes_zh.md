# 配方图鉴

## 敌对生物掉落物：
黄绿色染料 + 蜂蜜瓶 = 史莱姆球  

- ⚠️注意：这个配方是有序的，要将黄绿色染料放在史莱姆球上方。  

橙色染料 + 木棍 = 烈焰棒  
淡蓝色染料 + 木棍 = 旋风棒  
白色染料 + 木棍 = 骨头  
白色染料 + 铁粒 = 恶魂之泪  
紫色染料 + 鹦鹉螺壳 = 潜影壳  
红色染料 + 干海带 = 腐肉  
黑色染料 + 甜菜根 = 蜘蛛眼  
棕色染料 + 干海带 = 幻翼膜  
灰色染料 + 糖 = 火药  
橙色染料 + 蜜脾 = 树脂团块  
染料 + 菌光体 = 蛙明灯  
紫色染料 + 附魔之瓶 = 龙息  

## 特定结构中获取的物品：
粉红色染料 + 野花簇 = 粉红花簇  
淡蓝色染料 + 紫水晶 = 海晶沙砾  
淡蓝色染料 + 燧石 = 海晶碎片  

## 友好生物掉落物：
白色染料 + 干海带 = 羽毛  
白色染料 + 藤蔓 = 线  
橙色染料 + 大型蕨 = 皮革  
黑色染料 + 蜜脾 = 墨囊  
蓝色染料 + 萤石粉 + 墨囊 = 发光墨囊  

## 方块：
棕色染料 + 苔藓块 = 泥土  
黄色染料 + 砂土 = 沙子  
灰色染料 + 砂土 = 沙砾  
黄色染料 + 干海带块 + 干草块 = 海绵  
棕色染料 + 灌木丛 = 枯萎的灌木  

## 转换：
红色染料 + 沙子 = 红沙  

## 其他：
品红色染料 + 青色染料 + 黄色染料 = 3黑色染料
