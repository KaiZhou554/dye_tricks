# Dye Tricks 

English | [简体中文](README_zh.md)

📖 Full recipe list: [recipes.md](recipes.md)  

A behavior pack for **Minecraft Bedrock Edition** that allows players to convert raw materials using dyes.

When building redstone machines in Minecraft, I realized that while honey blocks can replace slime blocks, sticky pistons still require slimeballs — which can be a real hassle.

So I thought: what if honey could be dyed to match slimeballs?
**Dye Tricks** is an add-on built on ideas like this, featuring dozens of clever crafting recipes.

With this mod, you no longer need to fight mobs. Simply grow plants, mine, and farm to obtain most mob-drop materials — and every recipe is fully renewable.

![Demo Image](docs/demo.webp)

## Features

- 🎨 Dye-based conversion system  
  Dyes act as the core catalyst to transform common materials into resources that match their color and texture, with intuitive recipes.

- ⚔️ No mob combat required  
  Many mob drops can now be obtained through farming, gathering, and processing, ideal for peaceful or automation-focused gameplay.

- 🏛️ No structure hunting  
  Some items normally locked behind rare structures can now be crafted with dye recipes, reducing exploration grind.

- 🐾 No animal hunting  
  Animal drops are replaced by plant or natural-material alternatives, enabling sustainable and farmable production.

- ♻️ Fully renewable resources  
  Several limited or non-renewable blocks can be recreated through conversion, making world resources renewable.

- 🧪 Creative & decorative expansion  
  Adds dye-driven utility and decorative conversions that expand creative building and customization options.

## Installation 
1. Go to the Releases page to download the latest version.
2. Import the behavior pack into Minecraft Bedrock Edition.
3. Enable the behavior pack in the world settings.
4. Ensure the world has Script API enabled.
